> 早就写好这些，一直在犹豫。真的决定要发给你了，这一天都在慌。想了想还是觉得自己很冒失，但写都写好也纠结这么久了，还是该给自己个交代把心里想的告诉你。我知道很唐突，希望你见谅。

在公司有点坐立不安。

从年初与你相见，也有四五个月。
你留给我满心好感。

时不时想起你，也时不时告诉自己不要多想。
今天路上刷朋友圈，看到你说，“我需要的是信仰的支柱，梦想的驱动力“。
说不上为何，心中又燃起火焰。可能我就是觉得自己可以。

一直碰到女生，心里泛起年轻的冲动、爱慕。
但见过你，一种从未出现的感觉，我始终念念不忘。

合适。我强烈的感觉，抑或只是我一厢情愿的相信，我们很合适。

这两年我愈发觉察，自己在感情方面，变得如此木讷。
害怕与陌生人接触，不愿花时间建立联系。心中期盼感情，又怯于追求，眼看着错过，却停步不前。
我不惧怕现实的沉重，但多少胆怯有天会被矫情、怯懦这些生命之轻压垮。

这是我做的不好的地方，让自己淹没在通讯录的默默无闻中。
但相对自身，我并没有站在原地，等着生活的大雾将我埋葬。

我想把自己铺开来展现给你，苦于找不到合适的办法。
我渴望让你了解我，又怕你不会给那么多时间。
写下这封信，冲动大过理智，但请你相信我心中的坦诚。

希望我的唐突不会惊扰到你。

- - -

说说我对自己的认识。原谅我慢慢去讲。


从小到大，我一直都是‘老好人’，五六年级的王丽洁就这么说我。虽然当时不觉着是好话，但也不反感，毕竟到高中我还是人缘很好，身边很多朋友（我一直搞不懂的是，为什么你们二班的人看我都比较怪？就算到后来，跟牟壮、殷啸宇也是大学接触比较多他俩才对我有认可。当然，可能本身我就很一般，自己没反省到位）。到了大学，我也顺利当上班长，参加各种学院、学校活动。自己虽然不明白是为了什么，但好像这都是‘应该’争取的。一切很顺利，在我未对这个世界建立感知的时候，被父母、被身边人、被外界环境，推着走到这步，也许自己还是幸运的。但，总有美中不足。

刚上大学我的专业是“特殊教育（心理学方向）”，这么诡异个名字肯定是没人听过。当初报志愿失误，提前批报了华东师大，而且误填了服从调剂。虽然这是个号称全国第一的专业，但老妈当时拿到通知书也没敢第一时间告诉我。后来到大一，很长一段时间，打打电话，老妈就会哭起来，说起当初报考时没有仔细检查，才有了不如意的结果。

就这样，我的大学从一个阴差阳错开始。其实后来直到大一结束，看到一份大学排名的榜单我才恍然发现华师要排到20多名，但华师每年录取的分数让我一直身处错觉相信这个学校是在十名左右。无奈，命运总是悄然无情。但又能怎样，只好硬着头皮往前走。回想面对高考，貌似自己能做的只是努力付出做好准备，但结果，也许只能听天由命。

整个大一，我都在担心、茫然中渡过，不知道四年后会是怎样出路。每到空闲我都止不住问自己，继续这个sb专业，未来在哪？我该强弩自己喜欢上毫无兴趣的东西吗？如果现在的处境我不想要，那怎样才好？而且，即使自己能走在理想的轨道，那到底，什么是理想，生活是为了什么？

一年的时间，我想到这些：

根本上，我活着、做事，是为快乐、开心。

+ 首先，不为别人，是自私地为自己。当直面内心，很多时候都免不了自私（当然这个自私只是想法上。世事复杂，即使跟最好的朋友也难免会有自私的想法，但去不去做才是人和人差别）。为他人还是自己，只为区别哪个是根本。它们不矛盾，很多时候为别人付出，也是给自己交代，让自己开心。

+ 相比需要漫长痛苦付出取得成就换来的幸福，我更看重平实点滴。努力在生活细节发现乐趣，才是快乐源泉。珍惜珍视，才能让自己长久处于快乐之中。

不再盲目、靠惯性、不明所以地做事。凡事都要有自己判断，怀疑、思考，明确自己想要。做了决定，在尽量不影响别人的情况下，努力坚持自己。

+ 大一入学就考了四级。后来的六级，一直不觉着这个证书多大意义，跟我的专业也基本不相关。所以到最后，也没去考六级。
+ 大一时候我是班级第一批入党的两个人，党校的课也已经考过拿到证书。后来要到新的学院要我转关系，拿到从原来学院开出的证明，忽然疑问自己入D到底是为了什么？我爱国但不爱D，这张纸唯一的作用可能就是评奖或者毕业时有点作用，但这也不可能是D要我加入的初衷。后来这张证明被我扔掉，我也再没怀疑过。

找到自己的兴趣。做自己想做，才能做好，也才能快乐。

+ 大一结束时，没和任何人商量，我去申请了转专业改学计算机。成绩达标面试通过后，学院告诉我文科转理科要降一年级。我想了很久，衡量是否该用一年时间去修正人生暂时错误的方向。同意后，告诉爸妈，他们吃惊的接受，也愿意相信这是个正确的选择。我始终相信，这是人生道路上第一次自己决定命运，想清楚利弊，在越来越多没有是非的选择上，给出自己的答案。

保持积极的心态，不被任何事击垮。

+ 诚实，心胸坦荡才无所畏惧。大学后半段我挂了二十多门课，到最后一学期，还有八门课要过。但从大学开始到最后惊险毕业，不会我可以交白卷，但我可以肯定所有考试都是真实水平，没有抄袭。我愿意相信，如果小的诚实我坚守不住，那某天必会在大问题上摇摆不定。原则问题的犹豫，就是给犯错提供空间，我不想也不该让自己走到那步。

+ 自信。我父母，包括这个社会好多人，喜欢将自信与张扬挂钩，就像他们把成熟与圆滑世故画等号一样。自信确是我之前没有的，长时间的对立谦逊与自信，让我从来不知何时该自信，何时该谦逊。但当我想明白这些，我便愿意在自己付出足够的努力后自信起来，让自己在不断的抉择中坚定自己。

在我看来，独立、诚实、自信给了我积极乐观的心态，它们相辅相成让我成长太多。这些，也是我独立思考后最重要的改变。

关于朋友。

+ 回想从前，好像认识很多人，但也多是点头之交。想到这些，我只觉浪费时间。
+ 相比认识新朋友，我更愿意花时间与好朋友处的更好。
+ 其实我不愿把任何认识的人都叫朋友，在心里能配得上这个词的，绝对是两个手数的过来的。这些人，每个都是我心中的骄傲。

有一点特别要说，

当我严格的树立自己很多看法后，有那么段时间总会和看到的、听到的、身边的很多朋友，有想法上的冲突。当我变的怎么想，就让自己怎么去做不在乎别人看法后，我挺庆幸自己比较及时认识到：这些，都只是自己的想法。我可以拿自己的尺去衡量世界，但绝不能以此要求别人，尤其朋友。所以我很明确，这些条框我只画给自己，对朋友的一切，我不会变。


当我一点点想清楚这些后，心中便不再痛苦与迷茫。好像人生第一次真正体会到，我可以清楚、明晰地看待这个世界。

- - -

当我降级开始第二个大一时，好像满血复活。走在自己选择的道路，上自己感兴趣的课，做自己喜欢的事。这一年我比大一努力，成绩也还好，竟然偶尔会有 A 出现。但是，当在自己喜欢的路上越走越远，一切从满心期待变成循规蹈矩后，新的问题接踵而至。

我不知道别人是否也会有同样的幻觉：时常我们抱怨，这不是我想做的，做不好也正常。要是让我做 xxx，我一定能做好。

非常无奈，这个陷阱困惑了我好一阵子。当真正面对自己选择、自己喜欢的东西时，迎面而来的只有生疏与无知。我已经告诉自己要踏实，把选择的路走好。但现实的问题已不再是我该做什么，而是怎样把一件感兴趣的事情做好？

很难。两件事情，我越发认识：

1. 成长过程面对的人群注定是更加优秀的群体，感觉良好的自觉聪明只会越来越虚幻，取而代之的是总有人更聪明又更努力的事实。
2. 运动、考试、玩游戏，那些偶尔为之的东西，可以靠小聪明、靠运气。但要做好，要提高能力，只能一点一滴日积月累。

坚持，纯粹的积累，不断的努力，是我对自己问题的回应。

当我反思自己做的不好的时候，例如私底下问过自己，为什么考上清华北大的不是我？我能想到的是，从小到大一直不错的成绩让我相信自己有那么点聪明，而考试的偶然成分又太多。当有过一两次自己连玩带学临阵磨枪，靠着小聪明和运气拿到不错成绩时，这种结果产生的幻觉是致命的。高一高二我一直在玩，我敢肯定，自己是班级最能玩的学生。但到了高三，各学科不再只是一小块知识的检验，每次考试都是综合前两年学习的成果，漏洞百出便是最痛的现实。当我真的告诉自己认真开始努力的时候，各处都是缺口，努力不起来的无力感到现在还记忆犹新。

渐渐发现，坚持会是伴我今后日子，任何事情中最难的一点。大二时候，想到自己从小爱运动，但从来都是短跑出色长跑不行，耐力始终是弱项。我报名参加了一次半程马拉松，我想让自己体会下，如果过程艰辛漫长，到底能不能完成。最后我咬牙跑了下来，虽然到终点时好像脚已经不是自己的了，但我确信自己得到了对坚持的信心。

坚持两个字，到现在我也不敢说自己能够做到，只能继续反思，不断尝试，努力让自己做好。

- - -

结束第二个的大一，计算机这门技能，我想自己已经上路。看清各种课程多数无用，不是没它存在的价值，是它或者与学科无关（那些蛋疼的公共课），或者太偏理论与实际脱钩。我选择只上自己喜欢的课，学自己认为有用的东西。吕占东在这方面对我影响很大。计算机中的算法是门透析整个系统的分支，包括我们看到的这个世界运转的方式，个中规律便是算法。几乎每个nb点的学校都会重视算法相关的一项竞赛，后来工作面试这也是每个公司关注的重点。当我还是个门外汉时，我便因为占东，选择做这门竞赛。到后来，把重心都放在这里，不在乎那些填充学分的选修课是否能过。最后，虽然成绩只能说一般，但我得到了很多真正动手结合实际的锻炼，也在找工作时得到很多便利。

大学，在我看来最根本的好处便是大把大把的时间，让我多能自由支配，无拘无束，做自己喜欢的事情。

+ 电影，是我大学时光最大的收获。也许开始只是消遣时光，但我清楚地记着当大一上学期结束，夜晚窝在家看过《死亡诗社》那种透彻心扉的感觉，这让我十分坚定地认识到，电影真的能为我开启很多扇门。我对电影的态度是，有多认真对待就有多少回报。我愿把自己最清醒的状态留给电影，从不跳过某个片段或者落下某句台词，每个细节都是潜在铺垫，哪怕开场漫长的字幕，背景音也包含导演为全片选定的基调。我爱找到某个喜欢演员、导演的全部作品，一起看掉。每部电影看完，都去翻看影评，寻找与自己理解不同的地方。到大学毕业，去豆瓣标记了自己看过的电影，伴随我五年时光的应该有八百多部。到现在心底还一直期盼今后有天，能在条件允许的情况下，改行去做电影相关的事情。

+ 音乐。唱歌是我的弱项，每每跟朋友最不愿去的地方就是ktv，恨自己的确没有一副好歌喉。但我想唱歌与听歌是虽有共通的两件事吧。早期听歌一直都四处找专辑，硬盘塞的满满当当，后来有了网络平台，就常去虾米。刚去看了下，单从风格来论，我最喜欢的应该是摇滚和民谣。

    摇滚，国内的最爱是窦唯，其他有张楚、高旗、谢天笑、鲍家街、痛仰。国外的相对繁杂，Oasis，Nirvana，Travis，Manic Street Preachers，Fall Out Boy。The Black Keys 是我最近让我疯迷的乐队，他们最新的这张实在太赞，[Turn Blue](http://www.xiami.com/album/1895812468?spm=a1z1s.6928801.1561534521.2.j4n4RA)

    民谣，国内的爱留给李志、周云蓬、张玮玮。国外最近喜欢的有 Bon Iver，Of Monsters And Men，The Lumineers，John Grant。

    其他风格我比较中意 hip-hop，Eminem，Kanye West，还有去年风光无限的 Kendrick Lamar。

    我听歌的习惯是找到喜欢的歌手，听掉他的全部。所以这些喜欢的歌手我都是听过他们绝大多数基本全部的歌曲，才断定自己的喜爱。我还喜欢按专辑来听，这样最大的好处是，总能发现那些也许别人不喜欢但自己超爱的歌曲。每每找到那些躲在角落里的美妙旋律，好像发现宝贝，认定它们只属于自己。

    p.s. 这是我[收藏的歌手](http://www.xiami.com/space/lib-artist/u/463005)，不知是否也有你喜欢。

+ 书。相比电影和音乐，自己看书的步伐相对缓慢。上学的时候，语文就一直不好，从小也不爱看书。但是高中时候逐渐认识到书中的美好，偶尔看些小说。有次读到三毛的哭泣的骆驼，让我真切体会到了文字的力量，便陆玄看掉了三毛的全部文字。上大学，喜欢王朔、余华、村上、东野圭吾、野夫、冯唐。毕业这两年，告诉自己一定要坚持阅读，最近爱上王小波、苗炜、昆德拉和马尔克斯。

    p.s. [上班两年看掉的书单](http://book.douban.com/people/desperado.w/collect)

+ 我始终认为，书籍、电影、音乐，是我能认识到对心智最好的慰藉与磨砺。

+ 运动。篮球，我始终无法割舍，即使现在上班，也争取每周都打。高中时我是班级打球最好的，大学最后我是院队的主力，我们院最后拿到了学校的第二。足球，是我心底的最爱，虽然自小学基本就没碰过，但关注消息、熬夜看比赛，早已是生活的一部分。这两次世界杯，我都看了全部 3/4 的比赛。

想了想还有几件事要说：

+ 工作。在计算机院校，基本上就是两条好的出路，做项目和做竞赛。我选择了后者，同年级学院150多人，去做竞赛的只有5人。我们学院明显更看重相对功利的项目。这也让我自己产生了很多怀疑，必须承认这多少也源于自己意志的薄弱。很长一段时间，我基本不去上课，想让自己专心在竞赛，但不断面临的困难让我多少望而却步，一天的终点往往以失落与无尽的拖延结束。三四点钟，是我惯常的睡觉时间。但即使到了这个时间段，我也很难入睡。心中满是忧愁，不愿睡去结束又一个虚度的日子，也不愿睡去迎接新的一天。我真的担心自己的选择，害怕最后的出路。

    还好，一切还算顺利。开始找工作的季节连续奔波了10天左右，拿到心仪公司的offer，果断选择知足不再寻找。我得承认，在诸多领域中我能选择去一家做自己想做的技术的公司确实是幸运的。

+ 父母。每个人都庆幸自己拥有父母独特的爱，我也一样。当我回想成长的过程，相比于学校、书本、身边的环境，给自己最大的影响绝对来自父母。慢慢长大，越来越发现他们在自己身上留下的印记。大学有次晚出跑步，看到路灯下自己的身影，活脱是父亲走路的样子。真切认识到自己延续着他们的生命真是件无比奇妙的事情。虽然上学离家比较远，但也算经常回家，可能某天想家里买张车票就坐回了家。回想上大学到现在屈指可数的几次流泪，一次是爷爷去世，一次是看到一部特别契合我心的电影，再就是有次喝多了酒想念爸妈。

    与父母相处，自己心中最大的骄傲是与他们的沟通。在我能独立思考前，他们替我做决定，这之后，权衡他们的意见我来定。争取这个所谓的话语权，真是个漫长的过程。我一直在努力潜移默化地让他们相信，没有什么是不可以谈不可以商量的。一直以来爸妈都希望我考研，但我坚持认为计算机这个领域，实际经验比理论重要。刚上大三时候，有次和老妈打电话唠了四个小时，完全没有剑拔弩张的激烈，只是我们各自陈述自己的观点，一起权衡利弊。在面临越来越多不再是非对错那么简单的决定时，曾经的时代留给他们宝贵经验，但真正站在路口需要看清方向的是我自己。有些事情相信有他们的道理，但有些不是，就像现在的大学完全不是他们眼中的样子。我很庆幸爸妈最后选择相信我，我也在努力不让他们失望。

+ 感情。我从不信傻人傻福这种话，所有事情都该理智判断，冷静思考，但唯独感情，我自觉应该感性面对。感觉是很奇妙的东西，它不在天平的任一端，也许只在抛出硬币的一瞬间。

    我相信如果这世上所有其他事情都有所谓的技能，那爱情同样需要慢慢磨练。身边的朋友总有一部分碰到第一个便走入婚姻，我为他们的知足庆幸。但我想一旦错过第一个，就要经历更多。没人敢否认，知足是件多么困难的事情。我愿意等待，我愿意相信老罗那句“前方有个女孩等了我五百年”，但我也愿意知足，也许本来就没有完美，只是该如何让自己真正学会珍惜。

    我一直把婚姻当作是我需要面临最最重要的一个决定。它甚过曾经的高考与工作，因为它是另一个家庭的起点，是我余下生活的全部。

时光飞快。到世界杯结束，感叹这四年弹指灰间，到现在毕业上班也有两年。这是我这两年的一些反思：

+ 关于工作。我想自己这两年是努力的，比大学要努力。公司相对宽松的环境，给了我很多自由的空间，不再有人指点，全考自己摸索。做技术这条路永远都没有止境，付出越多就会走的越远。但我慢慢认识到，工作与生活是我需要平衡的地方。起初我一直把生活的中心放在工作之外，与朋友消遣，自己看书看电影。在公司只是把工作完成，多挣出些时间也是学些其他想学的东西。我试着把快乐建立在工作之外，让工作成为物质的支撑。但事实不是这样，自己也多少有些本末倒置。我渐渐意识到其实一天大半的时间都是在工作中渡过，如果不能把工作做好，那似乎便是某种程度的失败。我试着调整，让自己更加努力工作，试着去做到出色，在其中寻找快乐。

+ 关于知识。这个感觉越发强烈，要学的东西太多。工作上的，工作外的，单纯喜欢感兴趣的。我想如果说这么多年受的教育到底给了我什么，真的要说是对知识的肯定以及心中源源不断的渴望吧。从前莫名其妙的选择理科，那不是自己的决定，物理化一直是我的弱项，后知后觉才想到其实高一时候每次考试经常地理是各科的最高分。最近的日子，开始读历史，世界史、中国史，虽然只是起步，但纠正了自己好多愚蠢的误解。尝试去串起一条贯穿始终的时间线，有了好多全新的认识，这个过程美妙又快乐。

+ 关于如何平衡自身。从前很多事情，我都是抱着无畏的态度，其中就包括身体。抽烟、喝酒、作息不规律，我承认个中危害，也知道有天都会找回来，但我还是相信年轻的身体，Carpe Diem，无拘无束尽情享受。但上班两个月就胖了20斤的事实给我第一个警醒，长时间坐立办公让我的背部、颈椎开始疼痛，最不能忍受的是偶尔打球，运动半小时就呼哧带喘，完全找不到快感。所以，从去年七月我开始健身，十月左右开始游泳。这个过程，起初相对痛苦，难在需要坚持。但当一切趋于稳定，冬天到来我已慢慢适应在其中追求享受。一周中有两三天背着装备，下班走去附近的大学，锻炼完累成狗十点多到家，吃饭洗澡几乎将留给自己的空闲时间推迟到十一点开始。但我渐渐发现，愉悦身体是件多么开心的事，最最重要，这个过程让体会颇多。

    生活的愉悦，一方面来自内心的无比强大，另一方面来自身体的完好状态。生理总是在潜移默化促进好多改变。从开始锻炼，我好像生平第一次注意饮食，响应身体的需要，考虑哪些营养哪些健康。喝酒也好像一下过了愿意喝多的阶段，心情比酒重要，真的不愿喝多。同样，疲惫的身体有助于改善我长久不安的睡眠。最为重要，坚持锻炼的过程让我慢慢看清怎样才是生活该有的节奏。几乎现在需要面对的每件事情都要靠积累去完成，急不得也放不得，只能每天去完成一点一滴。锻炼给了我信心，让我的生活总在坚持些什么，并且在不断的努力中看到好的改变，也让我懂得放慢脚步，调整好身体和心态，一步步追寻想要的结果。
